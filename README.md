# A_Getter(CS) in C#

## Description
The program for getting links from specific websites. A very WIP version.

## Usage:
> dotnet ./A_Getter.dll [number of episodes] [link (in quotes)] (special eps?)

#### Supports for now:

 * http://okami-subs.pl/


### Project use: 

>.NET Core 2.0  
> HTML Agility Pack
